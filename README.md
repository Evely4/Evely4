### ✨ Meus Destaques

#### 📈 [Dashboard de Vendas Interativo](https://evely4.github.io/dashboard/)
> Projeto desenvolvido para análise visual de dados de faturamento.
> - **O que faz:** Exibe cards dinâmicos (Faturamento, Pedidos, Clientes), tabela de histórico mensal e um gráfico de linha progressivo (Chart.js) que garante crescimento constante.
> - **Funcionalidades:** Login fake para acesso, botão "Próximo Mês" para simular crescimento e "Zerar Tudo" para reiniciar a análise.
> - **Stack:** HTML5, CSS3, JavaScript.
> - *Foco:* Lógica progressiva, Chart.js e interface moderna.
